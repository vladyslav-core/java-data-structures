# Datenstrukturen (Java)
Java learning tasks from IT course (Austria).

## Aufgaben

### 1. Mengenlehre (HashSet)
- Berechnung von Vereinigungs-, Schnitt- und Differenzmengen
- Verwendung der Klasse `HashSet` und deren Methoden
- Unterstützung geschachtelter Methodenaufrufe  
  (z.B. A ∪ (B ∩ C))

### 2. Wörterbuch (Map)
- Bi-direktionales Wörterbuch (Deutsch ↔ Englisch)
- Hinzufügen und Entfernen von Wort-Paaren
- Effiziente Suche in beide Richtungen mit `Map`

### 3. Liste sortieren
- Eigene Implementierung eines Sortieralgorithmus
- Verwendung von `List` statt Arrays
- Keine Verwendung von eingebauten Sortiermethoden

## Projektstruktur
Jede Aufgabe ist in einem eigenen Package organisiert:
- `mengenlehre`
- `dictionary`
- `sorting`

## Hinweise
Dieses Projekt dient ausschließlich zu Lern- und Übungszwecken.
